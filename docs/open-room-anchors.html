What is GitHub
Think of it like
tracked drafts.
  • You know how you have draft 1, draft 2, draft 3 — and you can't remember what changed?
  • GitHub tracks every single change, meticulously
  • It also lets you work on a copy of the live site, so if something breaks — the real site stays up
  • Something will probably break today. That's normal. Even senior engineers break things.
Breaking something isn't a sign you can't do this. It's just part of it.
